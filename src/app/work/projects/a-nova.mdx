## Overview

Create a company that works remotely with clients in Lebanon. The company helps businesses and entrepreneurs achieve their technological visions.

## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **.NET**: For creating fast, scalable and reliable APIs.
- **Azure**: For fast cloud hosting and blob management.
- **Shopify**: For building reliable, modern e-commerce stores.
- **Firebase**: For building modern backend solutions (BaaS).
- **Flutter**: For building modern, cross-platform (Android and iOS) mobile applications.
